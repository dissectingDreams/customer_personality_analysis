# customer_personality_analysis
В этом репозитории я провел работу по анализу данных. Получив на вход сырую таблицу с информацией о покупателях, я попробовал вывести целевого потребителя для каждого вида продукции. Результаты проекта могут быть использованы для построения маркетинговой стратегии в компании.

# Цель работы :
  Выделить для каждого вида продуктов (вино, фрукты, мясо, рыба, сладости, золото) портрет целевой аудитории. То есть понять, какие клиенты покупают тот или иной продукт (по возрасту, доходу, семейному положению, количеству детей, каналу покупки).

Задачи :
  Очистить данные (пропуски, выбросы, удаление лишних признаков) ;
  
  Построить таблицу сравнения групп покупателей ;
  
  Визуализировать различные метрики из сравнительной таблицы

# Ключевые выводы / портрет покупателя
  Вино : Доход выше среднего, возраст 43+, высшее образование, женат/замужем. Вместе с вином часто покупает мясо и рыбу
  
  Фрукты : Самый высокий доход, возраст 42-43
  
  Мясо : Никак не отличается от общей выборки, вероятно, мясо покупают практически все
  
  Рыба : Похож на потребителя фруктов, но с меньшими тратами на сладости
  
  Сладости : Умеренный доход, высокие траты на золото
  
  Золото : Похож на покупателя мяса, однако чаще покупает вино

Общий вывод : покупатели любого продукта - это клиенты с доходом выше среднего, высшим образованием, возраст 42-44 года. В целом между группами довольно мало различий, в том числе по доходу. Наиболее "дорогой" сегмент - покупатели фруктов и рыбы.
