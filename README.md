# customer_personality_analysis
В этом репозитории я провел работу по анализу данных. Получив на вход сырую таблицу с информацией о покупателях я постарался вывести целевого потребителя для каждого вида продукции. Эта работа может быть использована для проведения маркетинговой политики в компании для более эффективного распределения ресурсов и увеличения доходности.
