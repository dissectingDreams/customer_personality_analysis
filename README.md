# customer_personality_analysis
В этом репозитории я провел работу по анализу данных. Получив на вход сырую таблицу с информацией о покупателях, я постарался вывести целевого потребителя для каждого вида продукции. Эта работа может быть использована для проведения маркетинговой политики в компании : оптимизации распределения ресурсов и увеличения доходности.

Цель работы :
  Выделить для каждого вида продуктов (вино, мясо, рыба, сладости и т.д.) портрет целевой аудитории. То есть понять, какие клиенты покупают тот или иной продукт (по возрасту, доходу, семейному положению, количеству детей, каналу покупки).
